# Метод автоматического машинного перевода текстов c вербального языка в последовательность глосс

## [English](README_EN.md)

Данная работа посвящена методу автоматического машинного перевода текстов в глоссы (переводы жестов в слова и словосочетания вербального языка).
В рамках метода был собран и размечен вручную параллельный корпус предложений на русском языке [ссылка](corpora/sentences_translations.csv).
Для разметки использовалась часть корпуса русского жестового языка Slovo [Kaggle](https://www.kaggle.com/datasets/kapitanov/slovo), файл annotations.csv [ссылка](corpora/annotations_slovo.csv), выложенный в открытый доступ.
Данный параллельный корпус использовался для обучения моделей трансформеров и больших языковых моделей, согласно предлагаемому методу.
